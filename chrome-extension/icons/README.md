# Chrome Extension Icons

This folder should contain the following icon files:

- `icon16.png` - 16x16px icon for toolbar
- `icon48.png` - 48x48px icon for extension management
- `icon128.png` - 128x128px icon for Chrome Web Store

## Creating Icons

You can create icons using:

### Online Tools
- [Favicon.io](https://favicon.io/) - Generate from text/emoji
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- [Canva](https://www.canva.com/) - Design custom icons

### Design Software
- Adobe Illustrator
- Figma
- Inkscape (free)

### Quick Method using Emoji

1. Go to https://favicon.io/emoji-favicons/shield/
2. Generate shield emoji icon
3. Download and extract
4. Rename files to icon16.png, icon48.png, icon128.png
5. Place in this folder

## Temporary Solution

For testing, you can use the provided `icon.svg` file. However, Chrome extensions require PNG format for production.

To convert SVG to PNG quickly:
1. Open `icon.svg` in a browser
2. Take screenshot
3. Crop and resize to 16px, 48px, and 128px
4. Save as PNG files
